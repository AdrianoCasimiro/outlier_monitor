# outlier_monitor
Detecção de consumo atípico

# Como rodar o projeto

- Clone o repositório
- Faça o build dos container
- Inicie os containers

```
git clone https://github.com/AdrianoCasimiro/outlier_monitor.git
docker-compose build
docker-compose up
```
